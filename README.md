# **Middleware**


## 功能
- 顯示request的時間戳記,方法,URL
- 顯示response的時間戳記,方法,URL
- 列出從收到req到送出res所花費的時間

![image]../../public/photos/time.png


## 如何使用

1.下載
    + 使用Terminal  輸入 git clone https://github.com/Csyhd/tiny-URL.git

2.進入
    + 使用cd 進入到下載完成的資料夾

3.安裝使用套件
    + npm install

4.執行
    + npm run dev 

5.執行成功
    + Terminal 顯示start

開始使用吧！
